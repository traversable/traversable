### new features

Adds a new module called `tree`, which is one that
    we're pretty excited about.
    
    With this PR, the following features are available on `tree`:
    
    - `tree.fromPaths`
    - `tree.toPaths`
    - `tree.has`
    - `tree.get`
    - `tree.set`
